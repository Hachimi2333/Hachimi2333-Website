# 前言

花了一下午的时间也是把博客从Hexo迁移至Fuwari了，使用了Twikoo给博客添加评论系统，但是效果不尽人意。

# 问题

Twikoo的评论系统中包含“点赞”、“回复”等按钮，点击这些按钮会导致自动跳转至页面顶部的问题。

原因出自Twikoo的官方版本对于这些按钮使用了包含链接`href="#"`的a标签。需要修改的方法也很简单，将`#`改成`javascript(0)`就可以了。

# 修复方法

## 复制仓库

```powershell
git clone https://github.com/twikoojs/twikoo.git
```

## 编译

```powershell
npm install --legacy-peer-deps
npm run build
```

很快就能在`dist`目录看到编译后的js文件了。

![编译](https://static.hachimi2333.top/blog/posts/12/dist.webp)

## 使用

将编译好的`twikoo.min.js`放入`public/twikoo`文件夹中，并在astro文件引用自编译的twikoojs文件。

```astro title="/src/components/comment/Twikoo.astro"
script.src = '/twikoo/twikoo.min.js';
```
